# Changelog

## v1.0.0 – Initial Release

### Overview
Dies ist das erste Release der **cher-prompt-generator** WebApp.  
Die Anwendung ist vollständig funktionsfähig, modular aufgebaut und bereit für Nutzung sowie Weiterentwicklung.

### Features
- 5 vollständig implementierte Prompt-Typen:
  - Start-Prompt
  - Drift-Korrektur
  - Rollenaktivierung
  - Persistenz-Prompt
  - Handover
- Dynamische Formularfelder basierend auf Prompt-Typ
- Live-Markdown-Vorschau
- Optionales HTML-Rendering über `marked.js`
- Copy-to-Clipboard Funktion
- Wizard-Modus (Grundstruktur)
- Responsive UI (Mobile & Desktop)
- Modulare JavaScript-Architektur:
  - `app.js`
  - `types.js`
  - `state.js`
  - `renderer.js`
  - `ui/*`

### Projektstruktur
- `index.html`
- `js/*`
- `assets/icon.svg`

### Qualität & UX
- Validierte Pflichtfelder
- klare Typografie
- optimierte Abstände

- UI-Polishing (Panels, Schatten, Layout)
